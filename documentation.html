    rapid_cost_estimate.cost_estimate.grid_search(configuration, device, input_size, train_loader, test_loader)¶

    Runs a grid search, designed to ascertain the optimum parameters for the network. The function is designed to be called a number of times, each time with the configuration altered slightly. Each call for the function produces a residual (which is recorded), with the parameters from the configuration that produced that residual also recorded.

    Thus, once each run of the grid search has concluded, a list containing the residuals, as well as a list with the corresponding values for the parameters, has been constructed.

    Parameters:

    configuration (dict): dictionary containing keys with associated values for the learning rate, hidden size of the network and the number of epochs.

    device (string): string describing what device is in use

    input_size (int): number of neurons needed in the input layer

    train_loader (torch.utils.data.dataloader.DataLoader): Dataloader that contains the data used to train the model.

    test_loader (torch.utils.data.dataloader.DataLoader): Dataloader that contains the data used to test the model.

    Returns:

    None
